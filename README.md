# Customer Churn Prediction

## Overview

Customer Churn Prediction is a Machine Learning project that predicts whether a customer is likely to leave a company based on customer demographics, account information, and service usage patterns.

## Problem Statement

Customer retention is crucial for business growth. This project helps identify customers who are likely to churn so that appropriate retention strategies can be implemented.

## Features

* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Machine Learning Model Training
* Customer Churn Prediction
* Interactive Streamlit Web Application

Install dependencies:

pip install -r requirements.txt

Run the Streamlit application:

streamlit run Customer_churn.py
